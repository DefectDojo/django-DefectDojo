You can attach image files to any Finding to provide visual context — for example, a screenshot of a vulnerability in action or a proof-of-concept image.

## Supported File Types

The following image formats are supported:

- `.png`
- `.jpeg` / `.jpg`
- `.gif`

## How to Attach a File

1. Open the Finding you want to attach a file to.
2. Click **Manage Files** within the Finding detail view.
3. In the file management view, select the file you want to upload and confirm.

The attached image will then be visible within the Finding.
