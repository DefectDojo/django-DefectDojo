Recomendaciones de Seguridad

Recomendaciones

  • Actualizar Apache XML Graphics Batik a la última versión estable que corrija esta vulnerabilidad.
  • Implementar una validación estricta de entrada para cualquier archivo SVG procesado por Batik.
  • Configurar un firewall de aplicaciones web (WAF) para filtrar y bloquear solicitudes maliciosas.
  • Limitar los permisos de la aplicación que utiliza Batik para reducir el impacto potencial de un ataque SSRF.
  • Implementar listas blancas para las URL permitidas en las solicitudes generadas por Batik.
  • Deshabilitar el procesamiento de entidades externas en XML si no es necesario para la funcionalidad de la aplicación.

            public boolean validateSVG(InputStream svgInput) {
                try {
                    SAXParserFactory factory = SAXParserFactory.newInstance();
                    factory.setFeature(XMLConstants.FEATURE_SECURE_PROCESSING, true);
                    SAXParser parser = factory.newSAXParser();
                    SVGValidator handler = new SVGValidator();
                    parser.parse(svgInput, handler);
                    return handler.isValid();
                } catch (Exception e) {
                    return false;
                }
            }
